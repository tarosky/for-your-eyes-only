# 翻訳更新コマンド

このコマンドはWordPressプラグインの翻訳ファイルを更新します。

## 手順

1. `composer i18n:update` を実行してPOTファイルを生成し、POファイルを更新する
2. `languages/fyeo-ja.po` を読み込み、未翻訳の文字列（msgstr が空のもの）を確認する
3. 未翻訳の文字列があれば、日本語に翻訳して PO ファイルを更新する
4. `composer i18n` を実行して MO と JSON ファイルを生成する

## 翻訳のガイドライン

- WordPress の一般的な用語は WordPress 日本語版に合わせる
- 技術用語はカタカナ表記を基本とする
- ユーザー向けのメッセージは丁寧語を使用する
- プレースホルダー（%s, %d など）はそのまま維持する

## 注意事項

- msgctxt（コンテキスト）がある場合は、そのコンテキストに応じた翻訳を行う
- fuzzy フラグがついている翻訳は確認して修正する
- PO ファイルのヘッダー情報は変更しない
