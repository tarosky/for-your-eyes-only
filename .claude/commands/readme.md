# README日本語化コマンド

このコマンドはREADME.mdを日本語に翻訳してREADME-ja.mdとして保存します。

## 手順

1. `README.md` を読み込む
2. 内容を日本語に翻訳する
3. `README-ja.md` として保存する

## 翻訳のガイドライン

- WordPress の用語は日本語版 WordPress に合わせる
- 技術用語はカタカナ表記を基本とする
- コードブロックやフック名はそのまま維持する
- リンクURLはそのまま維持する
- マークダウンの書式を維持する
- 見出しの階層構造を維持する

## 注意事項

- プラグイン名「For Your Eyes Only」はそのまま維持する
- Contributors、Tags などのメタデータ部分は翻訳しない
- Changelog のバージョン番号と日付はそのまま維持する
